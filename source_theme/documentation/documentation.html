Quick setup

Once downloaded, unzip the compressed folder to see the structure of (the compiled) Boomerang. You'll see something like this: 

    
    boomerang/
    ├── assets/
    │   ├── bootstrap/
    │   ├── ...
    ├── css/
    │   ├── global-style.css
    │   ├── global-style.less
    │   ├── themes.less
    │   └── buttons.less
    │   └── shortcodes.less
    ├── font-awesome/
    ├── images/
    ├── js/
    │   ├── jquery.wp.custom.js
    │   └── ...
    └── index.html
    └── about.html
    └── ...

Folder/File description

Folder/File name Description
assets This folder contains all Bootstrap files and all installed plugins.
css This folder contains all css and less files needed to create the template stylesheet.
font-awesome This folder contains all files necessary for the vectorial icons used in the template. -
js This folder contains all javascripts.
global-styles.css Represents the compiled version of the less files. To edit the stylesheet use less files to create new classes and when finiseh use a LESS compiler (ex: Crunch) to compile your work.
jquery.wp.custom.js Represents the main js file where all client event initializations happen. This is the file where you can creat client events, functions etc.

Styling and Coloring

Boomerang comes with 6 colors by default. Each color represents a theme that can be set as default. On your request we can add more colors if needed or you can create your own if you have some CSS/LESS skills.

Theme CSS class Details
Red wp-theme-1 -
Violet wp-theme-2 -
Blue wp-theme-3 -
Green wp-theme-4 -
Yellow wp-theme-5 -
Orange wp-theme-6 -

There are two ways of setting the main theme: on each page or with jQuery.


Manual: On each page

Add the theme class name (ex: wp-theme-1) on the body tag on every page of the site as bellow:

<body class="wp-theme-1"> … </body>

Client event: with jQuery

Second method is to open the jquery.wp.custom.js from js folder, find the Active theme section and replace the theme class with the one you want to set. It will automatically add the desired class on the body tag on each page this script is included. If you don't want this method remove the entire line of code.

$("body").addClass("wp-theme-1");

Layout options

Boxed or Fluid

Boomerang has 2 layout versions: Boxed or fluid. The fluid version is set by default. If you want to set the boxed version find the div wich has the wrapper class and apply next to it the class boxed, as below:

<div class="wrapper boxed"> … </div>

You can now set new backgrounds to your website. For that, check the Creating and changing backgrounds section.

Header

Boomerang comes with many types of header. In the near future we will create even more types to satisfy any need of your website. The header is formed of: top header and header (logo + navigation).

The top header is optional. You can show it or hide it, customize it as you need. We have integrated some functionalities such language select and cart preview.

The header is formed by the logo and the navigation. To change the logo just open the file, find the logo area and change the image path.
The navigation comes in two versions: left aligned and right aligned. The examples can be found in the header-example files from the package. Choose the one is best for your website and start from there. We have implemented an extra feature that allow you to insert a third level navigation.


Navbar stickyness

Many customers asked for this feature. Now you can set the navbar as sticky from jQuery. It allows the header to be visible all the time when scrolling. To activate this feature we created a jquery function in jquery.wp.custom.js and and called it on page load:

function enableStickyNav(){ $(".navbar").sticky({ topSpacing: 0, className: "navbar-fixed" }); }

If you want this feature deactivated, just remove the line of code that calls this function. you can find it at the top of the same files.


Navbar contrasted

A new helpful class is now available for styling the navbar. The navbar dropdowns have now 2 versions: white and dark. The white one comes by default. To activate the dark one you have to apply the navbar-contrasted class on the navbar container as below:

<div id="navOne" class="navbar navbar-wp navbar-contrasted" role="navigation"> ... </div>

Slide bar

The slide bar is a very cool features of Boomerang. You may be familiarized with this feature from mobile apps. It is a very nice and clean way to display important content and actions only when needed. It's a must have for small device resolutions.

This features can be activated/deactivated very easily. All animations are made with CSS3 and client events with jQuery. Boomerang comes with slide bar activated as default.

Search

We have implemented two types of search options. Choose the one that fits best with your website needs: navbar search or slide bar search.

Sections and boxes

Next, we will describe Boomerang content structures. We have implemented many features to make it easier to customize.


Sections

Boomerang has modular components for making these easier to re-use anytime when needed. That's why we splitted the content block into sections. Every new section is wrapped in a section which has the class slice.

<section class="slice"> ... </section>

The standard structure for creating content block is the next one: 

    
    <section class="slice">
        <div class="w-section inverse">
            <div class="container">
                <div class="row">
                    <div class="col-md-6"> 
                    	... 
                    </div>
                    <div class="col-md-6"> 
                    	... 
                    </div>
                </div>
            </div>
        </div>
    </section>


For setting a section background we created some css classes that can be directly applied on the section tag class. You can create even more background colors in the themes.less file. The default section background classes are:

Background CSS class Description
Base .bg-2 It applies the base color of the selected theme (red, blue, green etc). The base color can be changed by changing the value of the @baseBg variable.
White .bg-3 It applies a white color on the section. The white color can be changed by changing the value of @lightBg variable.
Dark .bg-4 It applies a dark color on the section. The dark color can be changed by changing the value of @darkBg variable.
Striped .bg-5 It applies a gray tone color on the section which is best if you want to create a striped layout by alternatig white and gray sections. It automatically sets a top and bottom border for better delimitation. The striped color can be changed by changing the value of @contrastBg variable.
